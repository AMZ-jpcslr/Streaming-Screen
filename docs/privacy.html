プライバシーポリシー

Streaming-Screen（以下「本サービス」）のプライバシーポリシーです。

0. 本サービスについて

本サービスのアプリ名（Google OAuth 同意画面の表示名）は Streaming-Screen です。

1. 取得する情報

本サービスは、YouTube Live のコメント表示のために、ユーザー（配信者）がGoogle/YouTubeアカウントでOAuth認可した場合に限り、YouTube Data API v3 を通じて次の情報を取得します。

  • YouTubeライブチャットのメッセージ内容（例：コメント本文、投稿日時）
  • 投稿者の表示名、およびチャット上の属性（配信者/モデレーター/メンバー等の表示に利用するフラグ）
  • スーパーチャット/メンバー通知の判定に必要な情報（APIが返す範囲）

本サービスが要求するOAuthスコープは https://www.googleapis.com/auth/youtube.readonly（読み取り専用）です。

2. 利用目的

  • OBSのBrowserSource上に、配信用オーバーレイとしてライブチャットを表示するため
  • 配信中の特別イベント（例：スーパーチャット）を画面表示するため
  • 動作状態表示（取得のON/OFF、エラー表示等）を提供するため

3. 保存・保管について

本サービスは、次のデータを保存する場合があります。

  • 表示設定：お知らせ文、X ID、ロゴURL、チャットiframe URLなどを settings.json に保存することがあります（運用者が保存機能を有効にした場合）。
  • セッション情報：ログイン状態の維持のため、Cookie（cookie-session）を使用します。

ライブチャットのメッセージ自体は、基本的に画面表示のために取り扱い、永続保存を目的としていません（ただし運用環境やログ設定により、サーバの標準ログに一部情報が出力される場合があります）。

4. 第三者提供

本サービスは、法令に基づく場合を除き、取得した情報を第三者に提供しません。

5. セキュリティ

本サービスでは、管理ページ（/control）をBasic認証等で保護する運用を推奨します。運用者は、認証情報や環境変数（OAuthクライアント情報）を適切に管理してください。

6. クッキーについて

本サービスはセッション維持のためにCookieを使用します。ブラウザの設定によりCookieを無効にすると、ログイン状態の維持ができません。

7. OAuth認可の取り消し・削除方法

  • 本サービス側では、/api/auth/logout（ログアウト）によりセッション情報を破棄できます。
  • Googleアカウント側での許可取り消しは、Googleの「セキュリティ」→「サードパーティのアクセス」等から実施できます。

8. 改定

本ポリシーは必要に応じて改定されます。改定後の内容は本ページに掲載した時点で有効になります。

Streaming-Screen

YouTube Live のチャットを取得して、OBSのBrowserSourceにオーバーレイ表示するための小さなWebアプリです。

プライバシーポリシー
利用規約

概要

  • 配信者が自分のGoogle/YouTubeアカウントでOAuth認可を行い、YouTube Data API v3 を使ってライブチャットを読み取ります。
  • 取得したコメントはサーバからSSEで配信し、オーバーレイ側（HTML）が表示します。
  • このアプリは youtube.readonly スコープのみを要求します（読み取り専用）。

問い合わせ

サポート窓口（メール等）は、運用者がOAuth同意画面で指定する連絡先に準じます。
